基于惯性大水滴滴水算法和支持向量机的验证码识别
==========

Inertial big drop fall algorithm, libsvm
------------------------

### 根目录目录结构
    |—— JavaVerify
    |   |—— download 从网上下载下来的原验证码图片，未经过处理
    |   |—— 1_gray 灰度化处理过的图片
    |   |—— 2_cfs 使用CFS连通域切割之后的图片
    |   |—— 3_drop 使用滴水算法切割之后的图片
    |   |—— 4_scale 缩放成16X16像素之后的图片
    |   |—— svm 使用livsvm建立的训练集，模型，以及分类的结果都保存在这里
    |   |—— src java的源代码
    |   |—— 其它的目录都是用来测试的
